# Create the Project Folder and Files

To start, you will create a new folder, `sinatra-voice-app`:

```shell
mkdir sinatra-voice-app
```

Inside the folder create two files: `server.rb` and `.env`

```shell
touch Gemfile
touch server.rb
touch .env
```
