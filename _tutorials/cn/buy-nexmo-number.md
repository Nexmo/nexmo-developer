购买 Vonage 号码
============

使用 Dashboard
------------

首先，可以浏览[您的现有号码](https://dashboard.nexmo.com/your-numbers)。

如果您没有备用号码，可以[购买一个](https://dashboard.nexmo.com/buy-numbers)。

使用 Nexmo CLI
------------

您可以使用 Nexmo CLI 购买号码。以下命令用于购买美国的可用号码。指定[两个字符的备用国家/地区代码](https://www.iban.com/country-codes)购买其他国家/地的号码。

    nexmo number:buy -c US --confirm

记录返回的号码，以备稍后需要时使用。
