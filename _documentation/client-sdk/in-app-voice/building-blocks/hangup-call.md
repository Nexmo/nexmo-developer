# Hangup a Call

In this building block you will see how to hangup a Call.

## Example

```building_blocks
source: '_examples/client-sdk/in-app-voice/hangup-call'
application:
  type: rtc
  name: 'Hangup Call'
```

## Try it out

When you run the code you hangup the Call.
