# Mute myself in a Call

In this building block you will see how to mute a Call.

## Example

```building_blocks
source: '_examples/client-sdk/in-app-voice/mute-myself'
application:
  type: rtc
  name: 'Mute Myself'
```

## Try it out

When you run the code you are muted in a Call.
